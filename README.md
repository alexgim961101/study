노트 작성·프로젝트 배치 규칙은 [AGENTS.md](./AGENTS.md)를 따른다.

## 목차

- [backend](./backend) — 백엔드 (서버, API, DB, 인증 등)
- [infra](./infra) — 인프라 / DevOps (배포, 컨테이너, 클라우드, CI/CD 등)
- [cs](./cs) — 컴퓨터 사이언스 기초 (네트워크, OS, 자료구조 이론 등)
- [algorithm](./algorithm) — 알고리즘 / 문제 풀이
- [languages](./languages) — 프로그래밍 언어 문법·특성
- [projects](./projects) — 독립 실행 미니 프로젝트
